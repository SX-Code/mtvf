<h1 align="center">MTVF</h1>

<p align="center">一款基于 Flutter 开发的影视资源阅览与播放应用，支持用户登录注册、影视收藏等功能，为用户提供便捷的影视观看体验。本应用所有资源均收集于互联网，仅用于学习交流，非盈利用途。如有侵权或商务合作需求，请及时与我联系。</p>

## 支持平台

- Android
- iOS（需要自签名）
- MacOS（待支持）
- Windows（待支持）
- Linux（待支持）
